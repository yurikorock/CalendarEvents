# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is currently not compatible with SWC. See [this issue](https://github.com/vitejs/vite-plugin-react/issues/428) for tracking the progress.

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

Це був такий день, коли в тебе все по-накатаному... Думки мої були зовсім заплутані і заважали одна одній, а я намагався їх вгамувати та впорядкувати.

І як це буває в книжках чи то у фільмах -  мені дзвонить один мій знайомий. 
Та і каже: " Привіт, є для тебе робота. Хочеш мені допомогти з одним проектом ?". 
Мої думки в цей час: "Як я маю йому допомогти? В нас різні сфері діяльності...".  
Та він продовжує: "В мене є один проект, але я не встигаю його зробити. Мені треба допомога, щоб хтось зробив менш відповідальну роботу, а я вже основні питання буду закривати."
Я йому у відповідь: "Окей, давай розповідай", а сам думаю: "Ну, відмовитись я завжди зможу, послухаю що від мене вимагається". 

Чи від того що він був впевнений, що я це зможу, чи то моя цікавість зробити щось невідоме для мене, в кінці кінців, після довгої розмови, я погодився йому допомогти. Практично я вже ступив "однією ногою" в ІТ- індустрію, сам того до кінця не усвідомлюючи куди це мене заведе.
 
В мене був час - місяць до старту проекту.
Я отримав посилання на вивчення матеріалу щодо діючого проекту (а нам треба було його редизайнити та обновляти логіку роботи додатку). 
Паралельно я шукав матеріали для вивчення зовсім незнайомої мені мови - мови програмування С#.
Я тоді себе відчував начебто мандрівник, який попав на невідому територію, де йому необхідно було вивчити мову, якою розмовляють мешканці.

Одразу скажу - це виклик для мозку, ще й який!!!
Прийшлося перелаштовувати своє мислення, включати всі свої резерви логіки на повну!

Не буду описувати, що далі відбувалось (лонгріди мало хто дочитує до кінця) :)))
Хоча там є що порозповідати і дуже багато цікавого.
Але затягнуло мене майже на півтора роки моєї участі з даним проектом.
Але все закінчується, проект закінчився - так і моя участь теж.

І ось що я відчув, вже після закінчення проекту - мені цього не хватає.
Тобто мене це зачіпило!!! Я не думав про це коли працював над проектом, я не думав коли вивчав документацію проекта чи матеріали, просто був в потоці і все. А тут, коли цього вже немає - мені вже не вистачало (і я не про те - типу щось кодити чи закривати якісь таски). Мені не вистачало чогось більшого, а саме що результати того, що я створив, мають сенс - приносять користувачу якусь цінність.

Але історія на цьому тільки розпочалась...
